## ✨ Características

- 🎨 **Interface Moderna**: Design premium com gradientes vibrantes e animações suaves
- 📝 **Organização Inteligente**: Sistema de pastas e categorias para suas anotações
- 🔔 **Lembretes**: Nunca mais esqueça de suas tarefas importantes
- 💡 **Gestão de Projetos**: Capture e desenvolva suas ideias criativas
- 🔍 **Busca Avançada**: Encontre rapidamente qualquer anotação por título, conteúdo ou tags
- 📱 **Responsivo**: Funciona perfeitamente em desktop, tablet e mobile

## 🚀 Tecnologias

Este projeto foi construído com as mais modernas tecnologias:

- **Frontend**: React 18 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS + shadcn/ui
- **Roteamento**: React Router
- **Estado**: React Query (TanStack Query)
- **Ícones**: Lucide React
- **Deploy**: Docker + Nginx + EasyPanel

## 🛠️ Desenvolvimento Local

### Pré-requisitos
- Node.js 18+ 
- npm ou yarn

### Instalação

```bash
# Clone o repositório
git clone https://github.com/juliolopes-dev/ideiaflow.git

# Entre na pasta do projeto
cd ideiaflow

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```

A aplicação estará disponível em `http://localhost:8080`

## 📦 Build de Produção

```bash
# Gerar build otimizado
npm run build

# Visualizar build localmente
npm run preview
```

## 📱 Funcionalidades

### 🏠 Dashboard
- Visão geral de todas as anotações
- Estatísticas em tempo real
- Cards informativos com métricas

### 📁 Sistema de Pastas
- Organize por projetos ou temas
- Cores personalizáveis
- Gestão completa (criar, editar, excluir)

### 📝 Tipos de Anotação
- **Notas**: Anotações gerais e ideias
- **Lembretes**: Tarefas com data de vencimento
- **Projetos**: Ideias de projetos e desenvolvimento

### 🔍 Busca e Filtros
- Busca por texto completo
- Filtros por categoria
- Filtros por pasta
- Tags personalizadas

## 🎨 Design System

O IdeiaFlow utiliza uma identidade visual moderna com:

- **Paleta de Cores**: Gradientes azul → roxo → rosa

- **Animações**: Micro-interações suaves
- **Componentes**: shadcn/ui customizados
